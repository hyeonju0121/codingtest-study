## 문제

## 출력
<p>첫째 줄에 다음 세 가지 중 하나를 출력한다.</p>

<ul>
	<li>A가 B보다 큰 경우에는 '<code>></code>'를 출력한다.</li>
	<li>A가 B보다 작은 경우에는 '<code><</code>'를 출력한다.</li>
	<li>A와 B가 같은 경우에는 '<code>==</code>'를 출력한다.</li>
</ul>
